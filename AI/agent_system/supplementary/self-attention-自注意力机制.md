# self-attention｜自注意力机制

> ✏️ The self-attention mechanism can help models learn and understand the dependencies between elements in a sequence. By calculating correlation scores between elements, the self-attention mechanism weights information from different positions to more effectively capture important features in the sequence.

---

## Attention

一个典型的 Attention 思想包括三部分：**Q**（query）、**K**（key）、**V**（value）。

- Q 是 query，是输入的信息；
- key 和 value 成组出现，通常是原始文本等已有的信息；
- 通过计算 Q 与 K 之间的相关性 a，得出不同的 K 对输出的重要程度；
- 再与对应的 V 进行相乘求和，就得到了 Q 的输出；

> K 和 V 等价，它俩指的是一个东西。

**步骤：**

1. **计算 Q 对每个 K 的相关性（相似性）**，即 $F(Q,K)$；这里计算相关性的方式有很多种，常见方法比如有：
   - 求两者的【向量点积】。
   - 求两者的向量【余弦相似度】。
   - 引入一个额外的神经网络来求值。
2. **对 step1 的注意力的分进行归一化**；softmax 的好处首先可以将原始计算分值整理成所有元素权重之和为 1 的概率分布；其次是可以通过 softmax 的内在机制更加突出重要元素的权重；即为 value_i 对应的权重系数;
3. **根据权重系数对 V 进行加权求和**，即可求出针对 Query 的 Attention 数值。

自注意力是注意力机制的一种扩展，它使得网络能够更有效地处理序列内部的动态关系，这种能力使得模型在处理长距离依赖的任务中尤为有效，如在 Transformer 模型中所见。

**注意力 vs 自注意力的区别：**

- **关注点不同**：
  - **注意力（Attention）**：通常用于关联两个不同序列（如编码器-解码器架构中的源序列和目标序列）。
  - **自注意力（Self-Attention）**：专注于单个序列内部，通过分析序列内部的元素之间的关系来计算权重。
- **应用范围**：
  - **注意力机制**：可以用于任何需要模型聚焦输入的重要部分的场景。
  - **自注意力机制**：特别适用于需要内部元素间互相影响的表示的场景，如处理长文本或复杂句子结构。
- **性能和效率**：
  - **自注意力**：由于其并行计算的能力，特别适合大规模数据处理，而这在使用标准的注意力机制时可能会更加复杂和耗时。

## self-attention

### what:

**核心思想：** 计算输入序列中不同位置之间的关联度来获取全局上下文信息。

**特点：** self-attention 可以远距离的捕捉到语义层面的特征。

**输入输出：**

CNN 输入的是一个向量！而自 self-attention 的输入是一个**向量集**，而且向量的大小、数目都是可变的。

> 一个 token 对应一个向量。

一个向量集（即矩阵）对应一个句子或一个输入序列。

**self-attention**，输入一排向量 $X=[x_1, x_2, ..., X_n]$，输出另外一个**同样长度**的一排向量。自注意力机制会为每个输入向量 $x_i$ 生成一个新的输出向量 $y_i$。这个新的表示 $y_i$ 不仅仅是原始向量 $x_i$ 本身，它是一个**融合了整个序列信息**的向量。它通过加权求和吸收了其他向量 $x_j$ 的信息。权重（注意力分数）决定了哪些向量的信息被"吸收"得更多。

这个过程是并行的，这通常是通过**矩阵运算**实现的。输入向量被组织成一个矩阵，自注意力的得分计算可以通过矩阵乘法一次性完成。

> 自注意力机制 —> 通过矩阵运算定义 —> 可并行化计算任务 —> 利用 GPU 硬件加速 —> 高效的并行处理

### why：为什么要用 self-attention

> ✏️ 全连接网络无法有效考虑输入序列中向量的关系。

对于全连接网络，无法判断两个 saw 的词性。为了让全连接网络考虑更多的信息，需要把每个向量的前后几个向量都"串"起来，一起输入到全连接网络。但是输入给模型的序列的长度，每次可能都不一样。如果要开一个窗口把整个序列盖住，可能要统计一下训练数据，看看训练数据里面最长序列的长度。接着开一个窗口比最长的序列还要长，才可能把整个序列盖住。但是开一个这么大的窗口，意味着全连接网络需要非常多的参数，可能不只运算量很大，还容易过拟合。

**想要更好地考虑整个输入序列的信息，就要用到自注意力模型。**

## 位置编码｜positional encoding

自注意力层少了一个也许很重要的信息，即**位置的信息**。对一个自注意力层而言，每一个输入是出现在序列的最前面还是最后面，它完全不知道。举个例子，在做词性标注的时候，动词比较不容易出现在句首，如果某一个词汇它是放在句首的，它是动词的可能性就比较低，位置的信息往往也是有用的。

需要考虑位置信息时，就要用到**位置编码（positional encoding）**。位置编码为每一个位置设定一个向量，即**位置向量（positional vector）**。位置向量用 $\bold{e}^i$ 来表示，把 $\bold{e}^i$ 加到 $\bold{a}^i$ 上面就结束了。

位置编码仍然是一个尚待研究的问题，在最早的 **"Attention Is All You Need"** 论文中，其位置向量是通过正弦函数和余弦函数所产生的，避免了人为设定向量固定长度的尴尬。

## 截断自注意力

> ✏️ 考虑输入序列中的小范围序列

**背景：** 在做语音识别的时候，把声音信号表示成一组向量，而每一个向量只代表了 10 毫秒的长度而已。所以如果是 1 秒钟的声音信号，它就有 100 个向量了，随便讲一句话都是上千个向量了。所以一段声音信号，通过向量序列描述它的时候，这个向量序列的长度是非常大的。在计算注意力矩阵的时候，其复杂度（complexity）是长度的平方。计算量就很可观，并且需要很大内存（memory）才能够把该矩阵存下来。

在做自注意力的时候，也许没有必要让自注意力考虑一整个句子，只需要考虑一个小范围就好，这样就可以加快运算的速度。**这就是截断自注意力。** 这个范围是人设定的。

比如：在做语音识别的时候，如果要辨识某个位置有什么样的音标，这个位置有什么样的内容，并不需要看整句话，只要看这句话以及它前后一定范围之内的信息，就可以判断。

## self-attention 和其他网络的对比

### self-attention 和 CNN

在 CNN 中输入的一维向量可以转为一个向量序列。比如 5×10×3 的图像转为每个元素是三维向量，共有 5×10 份向量的输入序列。

自注意力只要设定合适的参数，就可以做到跟卷积神经网络一模一样的事情。所以**自注意力是更灵活的卷积神经网络**。

> 卷积神经网络的函数集与自注意力的函数集的关系

**在做卷积神经网络的时候，只考虑感受野里面的信息。而在做自注意力的时候，会考虑整张图像的信息。**

在卷积神经网络里面，我们要划定感受野。每一个神经元只考虑感受野里面的信息，而**感受野的大小是人决定的**。而用自注意力去找出相关的像素，就好像是感受野是自动被学出来的，网络自己决定感受野的形状。网络决定说以这个像素为中心，哪些像素是真正需要考虑的，哪些像素是相关的，所以感受野的范围不再是人工划定，而是**让机器自己学出来**。

**随着数据量越来越多，自注意力的结果越来越好。** 最终在数据量最多的时候，自注意力可以超过卷积神经网络，但在数据量少的时候，卷积神经网络是可以比自注意力得到更好的结果的。

自注意力的弹性比较大，所以需要比较多的训练数据，训练数据少的时候就会过拟合。而卷积神经网络的弹性比较小，在训练数据少的时候结果比较好。但训练数据多的时候，它没有办法从更大量的训练数据得到好处。这就是自注意力跟卷积神经网络的比较。

### self-attention 和 RNN｜Recurrent Neural Network

目前，**RNN 很大一部分都可以用 self-attention 来取代了。**

在循环神经网络里面有一个输入序列、一个隐状态的向量、一个循环神经网络的块（block）。

> 循环神经网络中的隐状态存储了历史信息，可以看作一种记忆（Memory）。

循环神经网络的块"吃"记忆的向量，输出一个东西。这个东西会输入全连接网络来进行预测。接下来当第二个向量作为输入的时候，前一个时间点"吐"出来的东西也会作为输入丢进循环神经网络里面产生新的向量，再拿去给全连接网络。依次有序执行，这就是循环神经网络。

**self-attention 与 RNN 的比较：**

**对比一：**

- **RNN**：如果最后一个向量要考虑第一个向量的输入，就必须把第一个向量的输入存在记忆里面，一路带到最右边，才能够在最后一个时间点被考虑。
- **self-attention**：可以轻易地从整个序列上非常远的向量抽取信息。

**对比二：**

- **RNN**：无法并行，只能依次计算。
- **self-attention**：输入一组向量，输出的时候，每一个向量是同时并行产生的。所以在运算速度上，自注意力会比循环神经网络更有效率。

> 很多的应用已经把循环神经网络的架构逐渐改成自注意力的架构了。

## 优势

它们被广泛用于提高神经网络的可解释性，帮助解释模型的决策过程，使得原本被认为是黑盒模型的神经网络变得更易解释。这对于人们对机器学习模型的公平性、可追溯性和透明度的关注具有重要意义。
