# Go 概述

## 包（Package）

Go 有两种类型的包：

- **库包（Library）**：如 `package fmt`、`package net/http`，被其他包导入使用
- **main 包（Executable）**：`package main` 声明这是可执行程序，`func main()` 是程序入口

### 基本用法

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

- `go run hello.go` — 编译并立即执行，不留二进制文件，适合快速测试
- `go build hello.go` — 编译生成可执行文件，通过 `./hello` 运行

### init 函数

所有包都可以有 `init()` 函数，它在 `main()` **之前**自动执行，用于初始化操作（设置变量、连接数据库等）。

```go
package main

import "fmt"

var globalMessage string

func init() {
    fmt.Println("Running init()...")
    globalMessage = "Hello from init"
}

func main() {
    fmt.Println("Running main()...")
    fmt.Println(globalMessage)
}
```

### main 包由多个文件组成

同一目录下多个文件可以共同构成一个 `main` 包：

```
my_app/
├── main.go      // package main
└── helpers.go   // package main
```

`helpers.go` 中定义的函数可直接在 `main.go` 中调用（同包内无需导入）。

运行方式：

- `go run main.go helpers.go`（指定所有文件）
- `go run .`（推荐，自动找到当前目录所有 `package main` 文件）

### main 包导入自定义包

```
my_project/
├── go.mod        // go mod init my_project
├── main.go       // package main
└── greeter/
    └── greeter.go  // package greeter
```

```go
// greeter/greeter.go
package greeter

// 首字母大写 = 导出，可被其他包访问
func Hello() string {
    return "Hello from greeter package!"
}
```

```go
// main.go
package main

import (
    "fmt"
    "my_project/greeter"
)

func main() {
    message := greeter.Hello()
    fmt.Println(message)
}
```

运行 `go run .` 或 `go run main.go`，Go 会自动寻找并编译 `import` 的依赖包。

### 规则总结

1. **不能导入 `main` 包** — 它是程序终点，不是可复用的库
2. `**package main` 必须有 `func main()`** — 否则报错 `undefined: main.main`
3. `**func main()` 无参数无返回值** — 需要命令行参数用 `os.Args`
4. `**go run` 只需指定 main 包的文件**，import 的其他包由 Go 自动处理

## 命令行参数：os.Args

- 类型：`[]string`（字符串切片）
- `os.Args[0]` — 程序自身的名字
- `os.Args[1:]` — 用户传递的实际参数

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    fmt.Println("程序名:", os.Args[0])
    fmt.Println("参数:", os.Args[1:])
}
```

运行示例：

```bash
go run main.go hello world
# 程序名: /tmp/go-build.../main
# 参数: [hello world]
```
